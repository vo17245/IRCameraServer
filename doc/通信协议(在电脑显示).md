## 1.client hello
```
0: 
    0x00 -> protocol version
```

## 2.server reply
```
0: 
    0x00 -> OK<br>
    0x01 -> Failed<br>
```
在这之后客户端服务器建立连接，客户端开始向服务器发送数据

## 3.client send data
```
0-3(size): unsigned int -> dng data size<br>
4-(size+3): dng data<br>
```
## 4.server recv reply
```
0:
    0x00 -> OK<br>
    0x01 -> Failed<br>
```
在发送recv reply之后，服务器断开socket 连接
